# 🐭 Sneaky Mouse — Cheese Heist!

A cute 3D **stealth** game you play in your web browser — on a computer or a phone!

- 🧀 Grab all the cheese in each level, then sneak to the exit 🚪.
- 👁️ Each cat has a **vision cone** you can see — stay out of it!
- 🧱 Hide behind walls to break the cat's line of sight.
- 🐾 **Sneak** (hold Shift / the SNEAK button) to move quietly — slower, but much harder to spot.
- 🚪 Beat all **3 levels** to win.

### Controls
- 💻 **Computer:** arrow keys or W A S D to move, hold **Shift** to sneak.
- 📱 **Phone/tablet:** joystick to move, hold the blue **SNEAK** button.

Made with [Three.js](https://threejs.org). It's all in one file: `index.html`.

> A polished remake of the *mouse-vs-cat stealth* game prototype, rebuilt with a nicer design.
